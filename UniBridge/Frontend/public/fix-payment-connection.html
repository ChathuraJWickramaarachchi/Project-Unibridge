🔧 Fix Payment Connection Error

❌ Current Error: ECONNREFUSED

Issue: Vite frontend cannot connect to backend server at localhost:5001

Cause: Backend server is not running

🚀 Quick Fix - Start Backend Server

Step 1: Open Terminal/Command Prompt

Navigate to the Backend directory:

cd "c:\Users\Tumal\OneDrive\Desktop\Chathura\UniBridge\UniBridge\Backend"

Step 2: Start the Server

Run one of these commands:

Option 1: npm start Option 2: node server.js Option 3: npm run dev (for development with auto-restart)

Step 3: Verify Server is Running

You should see output like:

Server running on port 5001 ✅ Database initialization complete Loading routes... Routes loaded successfully

Step 4: Test Connection

🛠️ Alternative Solutions

If Server Won't Start:

1. Check Node.js Installation

node --version

If not installed, download from: https://nodejs.org/

2. Check Dependencies

npm install

This will install all required packages

3. Check Environment Variables

Make sure .env file exists with proper configuration:

PORT=5001 MONGODB_URI=mongodb://localhost:27017/unibridge JWT_SECRET=your-secret-key

4. Check MongoDB Connection

mongosh

If MongoDB is not running, start MongoDB service

🔍 Troubleshooting Steps

  1. Verify Backend Port: Make sure port 5001 is not blocked
  2. Check Firewall: Allow Node.js/localhost connections
  3. Restart Everything: Stop both frontend and backend, then restart backend first
  4. Clear Cache: Clear browser cache and restart frontend
  5. Check Logs: Look at backend console for specific error messages

✅ Expected Working State

When Everything Works:

  • ✅ No more ECONNREFUSED errors
